Ensayo: ¿Por qué usar HTML Semántico?

Autor: Raquel Villafuerte

Introducción

El HTML semántico es el uso de etiquetas que tienen un significado específico tanto para los desarrolladores como para los navegadores.

Desarrollo

Al utilizar etiquetas como <header>, <main> y <section>, ayudamos a que los motores de búsqueda como Google entiendan mejor nuestra página.

Esto también mejora la accesibilidad, permitiendo que personas con discapacidad visual usen lectores de pantalla con mayor facilidad.

Conclusión

En conclusión, aprender a maquetar correctamente es la base para ser una excelente Ingeniera de Sistemas y Desarrolladora Frontend.
